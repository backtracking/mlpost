type command =
| Fill_rect of info * float * float * float * float

Fill_rect info x y w h should draw a rectangle at (x,y) of width w and height h.

| Draw_text of text
| Specials of info * string * float * float

Specials info s x y should draw special s, encoded as a string, at pos. (x,y). info can contain additional information such as color.

| Draw_text_type1 of text_type1

Can appear only after a decomposition of text

type page = command list
val load_file : Mlpost.File.t -> page list

load_file arg fn loads the dvi document in file fn, passes arg and the loaded document to Dev.new_document and calls the drawing functions of Dev as needed. At the end, the return value of the device is returned. The command list are in reverse order inside a page
